# lab

Personal archive of early machine learning, computer vision, deep learning, and coursework projects (2020–2024).

These were learning exercises and small experiments — kept here for reference and to preserve the commit history of where I started.

For my current work, see my [pinned repos](https://github.com/harneet2512) — RobbyMD, GroundTruth, TracePilot, ExecutionDesk-AI, Codetune, council-mcp, and more.

## Contents

| Folder | What's in it |
|--------|--------------|
| `cv/` | OpenCV / computer vision projects |
| `dl/` | Deep learning notebooks (CNNs, GANs, transfer learning) |
| `ml/` | Classical machine learning |
| `nlp/` | Natural language processing |
| `streamlit/` | Streamlit web apps |
| `coursera/` | JHU / Coursera coursework |
| `cpp/` | C++ DSA practice |
| `misc/` | Other early projects |

Each subdirectory preserves the original repo's full git history.
